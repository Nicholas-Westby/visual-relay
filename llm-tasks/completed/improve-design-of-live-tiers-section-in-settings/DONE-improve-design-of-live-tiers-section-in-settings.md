# Improve Design of Live Tiers Section in Settings

The text is a bit small, the are multiple alignment issues, there is no vertical spacing between dropdowns.

The text should be the standard sizes used by the rest of the app, the horizontal alignment should be in good taste, and there should be some amount of vertigal gap between the dropdowns.

I'm not sure what the text to the right of each dropdown is (perhaps the API provider), but it doesn't seem to change on dropdown selection and isn't configurable, so it should just be removed entirely.
